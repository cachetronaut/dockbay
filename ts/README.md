# dockbay

TypeScript implementation of DockBay.

For product-level context, shared contracts, and cross-language repository information, see the public repository: https://github.com/cachetronaut/dockbay.

## Install

```sh
npm install dockbay
```

## Development

Run from `ts/`:

```sh
pnpm install
pnpm verify
pnpm build
```
